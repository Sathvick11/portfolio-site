## Climate Data Analysis Pipeline (CDAP)

`Python` · `scikit-learn` · `Pandas` · `Jupyter`

- Analysed historical weather station data from ~600 European stations for a university data science project.
- Built an end-to-end preprocessing pipeline covering missing value imputation, feature engineering, and normalisation.
- Implemented a baseline modelling ladder — Dummy Regressor → Linear Regression → Random Forest — to systematically benchmark model complexity.
- Evaluated generalisation using **leave-one-city-out cross-validation** to simulate real-world deployment on unseen locations.
